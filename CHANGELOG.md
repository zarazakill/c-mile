# Changelog

All notable changes to the C-mile project will be documented in this file.

## [0.9.5] - 2024-xx-xx

### Added
- Added progress tracking with speed and time estimation
- Implemented verification option after writing
- Added force writing functionality for protected devices
- Added cluster size configuration options
- Added device information display

### Changed
- Updated application name to "C-mile v0.9.5" in UI
- Improved error handling and user feedback
- Enhanced device detection capabilities
- Updated build system to use more modern Qt6 components

### Fixed
- Fixed potential crashes during device refresh
- Fixed issues with image file validation
- Improved memory management during write operations

## [0.7.4] - 2024-xx-xx

### Added
- Initial public release
- Basic image writing functionality
- Device detection and selection
- Graphical user interface
- Support for multiple image formats (ISO, IMG, etc.)
- Basic formatting capabilities
